How it works

Three steps. Under two seconds.

From spoken word to applied code change — every stage is deliberate.

01 — Capture
🎙
Voice to text
Audio runs on a dedicated AudioWorklet thread. Encoded as WAV, transcribed by faster-whisper (small.en, int8, CPU) in ~300ms. Fully offline.
02 — Classify
🧠
Intent detection
Logistic regression on sentence embeddings maps your transcription to one of 6 intents — EDIT, REFACTOR, GENERATE, EXPLAIN, NAVIGATE, UNDO — in under 10ms.
03 — Transform
✦
AI code action
Command + intent + editor state → Gemini Flash via async FastAPI. Returns a precise JSON edit shown as a real Monaco inline diff before anything is applied.
Features

Built for the voice workflow.

Not a chatbot with a code block. Every detail designed around spoken commands.

🔊
AudioWorklet capture
Dedicated audio thread — not the main JS thread. Zero dropouts, no buffering latency. Graceful fallback to ScriptProcessorNode for older browsers.
W3C standard · future-proof
💾
Session persistence
Code, language, and filename saved to localStorage on every keystroke (debounced 800ms). Refresh the page — everything is exactly where you left it.
Zero backend required
⌨️
Text command fallback
Type commands directly if you're not in a mic-friendly environment. The same ML + Gemini pipeline processes them identically. Space bar toggles recording.
Accessible · keyboard-first
▶
Command replay
Every command stays in your history with a replay button. Re-run any past voice command against the current editor state — no re-speaking required.

FastAPI + Uvicorn backend
Async Python. Gemini and Whisper wrapped in asyncio.to_thread() — event loop stays free during every API call.
faster-whisper speech
CTranslate2-optimised Whisper. small.en, int8 quantised, CPU-only. ~300ms latency, fully offline.
scikit-learn classifier ML
Sentence embeddings → logistic regression. 352 examples, 6 classes. Retrains from your command logs automatically.
Monaco Editor frontend
VS Code's engine in the browser. createDiffEditor() for real inline diffs, 80+ languages supported.
Gemini Flash Lite LLM
Structured JSON output — startLine, endLine, newCode, explanation. application/json MIME enforced server-side.
